# CineVerse

This project is a web application that allows users to search and explore movies using the OMDb API. Users can view movie details such as title, poster, release year, and ratings. The application provides an interactive interface for discovering movies easily.

## Objective
The objective of this project is to implement JavaScript concepts like API integration using fetch, dynamic data rendering, and array higher-order functions for searching, filtering, and sorting.

## API Used
- **OMDb API**
- **Example Endpoint:** `https://www.omdbapi.com/?i=tt3896198&apikey=b4b376f3`

## Features Designed for Milestone 2
- Dynamic data rendering from the OMDb API
- Loading states and error handling
- Fully responsive layout across all device sizes
- Fetching API data using JavaScript `fetch`
- A premium, dark-mode design emphasizing aesthetic quality

## Features (Planned for Future Milestones)
- Search movies by title
- Filter movies based on year or type
- Sort movies alphabetically or by year
- Add movies to favorites
- Dark mode / Light mode toggle

## Technologies Used
- HTML5 (Semantic Structure)
- CSS3 (Custom Properties, Flexbox, Grid, Animations)
- JavaScript (ES6+, Fetch API, DOM Manipulation)

## Project Setup
1. Clone the repository: `git clone <repository_url>`
2. Open the project folder.
3. Open `index.html` in your web browser or use a local development server like Live Server.
