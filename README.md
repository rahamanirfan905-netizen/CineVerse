#  Movie Explorer App

##  Project Description

This project is a web application that allows users to search and explore movies using the OMDb API. Users can view movie details such as title, poster, release year, and ratings. The application will provide an interactive interface for discovering movies easily.

##  Objective

The objective of this project is to implement JavaScript concepts like API integration using fetch, dynamic data rendering, and array higher-order functions for searching, filtering, and sorting.

##  API Used

* OMDb API
* Example Endpoint:
  https://www.omdbapi.com/?i=tt3896198&apikey=b4b376f3

##  Features (Planned)

*  Search movies by title
*  Display movie posters and details
 Filter movies based on year or type
 Sort movies alphabetically or by year
 Add movies to favorites
 Dark mode / Light mode toggle

## Technologies Used

* HTML
* CSS (or Tailwind/Bootstrap)
* JavaScript (Fetch API, Array HOFs)
